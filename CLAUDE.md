# 프로젝트: 나의 소유물 관리(mytem)

## 프로젝트 개요

인증된 사용자가 본인의 소유물을 체계적으로 관리할 수 있는 웹 애플리케이션

---

## 주요 기능

### 1. 소유물 관리 (CRUD)

- 소유물 등록, 조회, 수정, 삭제
- 소유물 속성: 이미지(픽토그램), 이름, 개수, 설명

### 2. 기본 픽토그램 이미지 제공

- 일반적인 소유 항목에 대한 기본 픽토그램 이미지 제공
- 예: 티셔츠, 신발, 책, 랩탑, 노트북, 가방 등

### 3. AI 픽토그램 이미지 생성

- 사용자가 텍스트로 원하는 이미지 설명 입력
- AI(나노바나나 등)를 통해 커스텀 픽토그램 이미지 생성
- 생성된 이미지를 소유물에 적용 가능

### 4. 키워드 검색

- 소유물 키워드 검색으로 기본 픽토그램 이미지 조회
- 검색 결과에서 선택하여 소유물 등록

---

## 주요 시나리오

### 시나리오 1: 기본 소유물 등록

1. 사용자가 회원가입 후 로그인
2. "책"을 키워드로 검색
3. 책에 대한 기본 픽토그램 이미지가 검색 결과로 표시
4. 이미지를 선택하고 책의 개수, 상세 정보 입력
5. 소유물 등록 완료

### 시나리오 2: 소유물 수정/삭제

1. 등록된 소유물 목록에서 "책" 선택
2. 상세 정보 확인
3. 개수나 설명 수정 또는 삭제

### 시나리오 3: 커스텀 픽토그램 생성

1. "가방"을 키워드로 검색
2. 원하는 결과가 없거나 기본 이미지가 마음에 들지 않음
3. 픽토그램 이미지 생성 메뉴 진입
4. 원하는 가방에 대한 설명을 텍스트로 입력
5. AI가 픽토그램 이미지 생성
6. 생성된 이미지로 소유물 등록

### 시나리오 4: 생성된 픽토그램으로 소유물 등록

1. 다시 "가방" 키워드 검색
2. 이전에 생성한 커스텀 픽토그램 이미지가 검색 결과에 포함
3. 해당 이미지 선택 후 소유물 정보 입력
4. 소유물 등록 완료
